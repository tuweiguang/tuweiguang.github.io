发表于 2019-09-05 | 分类于 Go

本文主要学习Go语言原生json编码包encoding/json使用。

JSON编码

函数声明：func Marshal(v interface{}) ([]byte, error)
v interface{}：Go语言的数据对象
[]byte：返回JSON格式字节数组
error：返回编码失败错误

json包只访问struct类型的导出字段并且以大写字母开头的字段，因此，只有结构体的导出字段才会出现在JSON输出中，若导出字段有tag，则JSON KEY显示tag名称。当然并不是所有的基本类型都可以编码(见下面代码)。注意：

  1. channel，complex和函数类型不能被编码
  2. 不支持循环数据结构
  3. 指针将被编码为它们指向的值（如果指针为零，则为“null”）

JSON解码

函数声明：func Unmarshal(data []byte, v interface{}) error
data []byte：需要解码的JSON格式字节数组
v interface{}：存放解码出对象的指针
error：返回解码失败错误

对于给定的JSON键“Foo”(首字母可以是大写或小写)，Unmarshal将查看目标结构的字段以查找（字段匹配按优先顺序如下）:

  1. 首先匹配：tag为“Foo”的导出字段（导出字段tag可以是大写或小写）
  2. 再次匹配：一个名为“Foo”的导出字段（导出字段名字首字母必须是大写）
  3. 最后匹配：名为“FOO”或“FoO”的导出字段或“Foo”的其他一些不区分大小写的匹配

Unmarshal将仅解码它可以在目标类型中找到的字段。如果JSON字符串有多余的字段将忽略，如果JSON字符串缺少部分字段将使用对应类型零值填充。
